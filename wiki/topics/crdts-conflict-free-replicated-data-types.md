# CRDTs: Conflict-Free Replicated Data Types

## Key Takeaways

- CRDTs provide eventual consistency in distributed systems
- They allow for availability and partition tolerance while avoiding complex conflict resolution logic

## Summary

CRDTs are data structures that can be replicated across nodes and updated independently without coordination, guaranteeing eventual consistency.
