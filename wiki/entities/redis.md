# Redis

## Overview

Redis is an open-source, in-memory data structure server used primarily as a cache or for maintaining real-time data. It supports various abstract data structures such as strings, hashes, sets, sorted sets, and lists. Redis provides high-performance capabilities through its in-memory storage mechanism.

## Key Details

- **Type:** In-Memory Data Structure Server
- **Primary Use Cases:**
  - Caching to speed up read operations
  - Maintaining real-time data for applications
- **Data Structures Supported:**
  - Strings
  - Hashes
  - Sets
  - Sorted Sets
  - Lists
- **Features:**
  - Persistence options (RDB and AOF)
  - Pub/Sub functionality
  - Cluster support
